## 🚀 Featured Projects

### ♟️ Yoki Chess Engine
> **C++** · [View repo](https://github.com/Sava2901/Yoki-Chess-Engine)

A from-scratch chess engine written in C++ featuring:
- Advanced position evaluation with piece-square tables and game-phase awareness
- King safety analysis, pawn structure evaluation, and passed-pawn detection
- Comprehensive Doxygen documentation across the entire codebase

---

### 🌐 Yoki Chess Web
> **TypeScript · React · Vite · TailwindCSS · Socket.IO · Supabase** · [View repo](https://github.com/Sava2901/Yoki-Chess-Web)

A full-stack chess web application powered by the Yoki engine:
- Real-time online multiplayer via Socket.IO
- OAuth authentication and avatar management with Supabase
- Offline mode with customisable AI opponents
- Time controls, pawn promotion UI, and move-history navigation
